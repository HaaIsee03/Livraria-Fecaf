Livraria FECAF

Projeto de Cadastro, Leitura, Edição e Exclusão de livros de uma Livraria. Front-End criado com Angular 15, Back-End criado com Java 8 + Spring Boot e o Banco de Dados H2 em memória. Criado para estudo de caso da faculdade FECAF na materia de Object Oriented Programming.

Projeto Back-End e Front-End

Back-End criado com Java 8 e Spring Boot + Banco de dados H2.

Front-End criado com Angular + Typescript.
